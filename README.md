Цель - отработать комплексное решение задачи машинного обучения на примете прогнозирования врмени поездки на такси в Нью-Йорке.
Задачи:
- освоить комплексное решение задачи по МОМО
- провести базовый анализ и расширене данных
- провести отбор признаков для модели
- составить нескоько моделей для решения задачи

Этапы поекта соответсвуют поставленным задачам.
Итг проекта: лучшие резульатаы оказала модель Catboost.
